What is this?

This is a RESTful API, which reads its data from the so called GAME_MASTER file. The data from GAME_MASTER gets transformed using the Github repository pokemongo-data-normalizer to a better processable state.

Contribute

This repository heavily relies on the help of the community. This API is splitted into 3 main repositories:

  • pokemongo-game-master - The raw GAME_MASTER data
  • pokemongo-data-normalizer - The transformed GAME_MASTER data to a better processable state
  • pokemongo-web-api - The final Web endpoint which exposes the transformed & processed data to the web
